### CSS Variables (Easy Customization)

All colors are defined as CSS variables in `:root`:

```css
--primary-color: #6366f1;           /* Main brand color */
--accent-purple: #a78bfa;           /* Purple accents */
--accent-cyan: #22d3ee;             /* Cyan accents */
--success-color: #10b981;           /* Success green */
--danger-color: #ff6b6b;            /* Error red */
--bg-dark: #0f172a;                 /* Dark background */
--bg-card: #1e293b;                 /* Card background */
--border-color: #334155;            /* Border color */
--text-dark: #f1f5f9;               /* Main text */
--text-lighter: #94a3b8;            /* Secondary text */
```
